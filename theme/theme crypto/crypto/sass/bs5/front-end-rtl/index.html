تلفن همراه

تمام قدرت صرافی ارز دیجیتال بایننس در کف دست شماست. همین امروز اپلیکیشن معاملات ارز دیجیتال موبایل Binance را دانلود کن.

دسکتاپ

پلت فرم معاملات ارز دیجیتال قدرتمند برای کسانی که قصد تجارت دارند. تجربه تجارت در بایننس، که برای دستگاه ویندوز یا MacOS شما ساخته شده و میتونی نسخه مورد نیازت رو دانلود کنی

API

Binance API طراحی شده تا راهی آسان و کارآمد برای ادغام برنامه ترید شما در پلتفرم ما ارائه بده.
